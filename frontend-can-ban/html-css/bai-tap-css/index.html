Thuộc tính HTML

Trong bài này chúng ta sẽ tìm hiểu về thuộc tính của thẻ trong HTML và làm quen với một số thuộc tính cơ bản

Thuộc tính HTML là gì?

  • Tất cả các phần tử HTML có thể có nhiều thuộc tính
  • Các thuộc tính cung cấp thông tin bổ sung về các phần tử
  • Các thuộc tính luôn được chỉ định trong thẻ bắt đầu
  • Các thuộc tính được quy định bằng các cặp tên / giá trị như: name = "value"
